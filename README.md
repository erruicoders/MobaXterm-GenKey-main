# MobaXterm-GenKey
你懂的！！

## 本地启动
需要安装Python3!!!
```
pip install --no-cache-dir -r requirements.txt
python app.py
```

## Docker
```
docker pull erruicoders/mobaxterm-genkey:latest
docker run -d -p 5000:5000 erruicoders/mobaxterm-genkey:latest
```
